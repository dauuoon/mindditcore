# Minddit Core 랜딩페이지 디자인 아이디어

## 아이디어 1: "Structured Serenity" - 미니멀리즘 + 구조적 우아함

### 디자인 무브먼트
**Bauhaus-inspired Minimalism with Soft Modernism**
- 기하학적 정확성과 인간적 온기의 조화
- 정신건강 서비스의 신뢰감을 구조적 정밀함으로 표현
- 과학적 엄밀함과 감정적 안정감의 동시 전달

### 핵심 원칙
1. **구조적 명확성**: 그리드 기반이 아닌 비대칭 모듈식 레이아웃으로 시각적 흥미 유지
2. **공간의 호흡**: 넉넉한 여백을 통해 각 요소가 독립적으로 존재하면서도 조화
3. **차분한 우아함**: 화려함 대신 정제된 디테일과 섬세한 타이포그래피
4. **신뢰감 있는 차가움**: 따뜻함과 전문성의 균형

### 색상 철학
- **Primary**: 딥 네이비 (oklch(0.35 0.08 260)) - 신뢰, 안정성, 전문성
- **Secondary**: 소프트 민트 (oklch(0.75 0.06 165)) - 희망, 회복, 신선함
- **Accent**: 따뜻한 회색-베이지 (oklch(0.88 0.02 70)) - 인간적 온기
- **Background**: 거의 흰색의 차가운 톤 (oklch(0.98 0.001 260)) - 명확성
- **감정적 의도**: 신뢰감 있으면서도 접근 가능한, 의료적이지 않은 현대적 전문성

### 레이아웃 패러다임
- Hero: 좌측에 텍스트, 우측에 추상적 기하학 요소 (비대칭)
- 기능 소개: 2x2 그리드가 아닌 캐스케이딩 카드 배치
- 강점 섹션: 좌우 교차 배치로 리듬감 생성
- 전체적으로 Z-pattern 시각 흐름

### 시그니처 요소
1. **구조적 라인**: 얇은 네이비 라인으로 섹션 구분 및 강조
2. **추상 기하학 패턴**: 원, 선, 사각형의 미니멀한 조합으로 "구조"를 시각화
3. **타이포그래피 강조**: "Minddit + Core"의 "+"를 기하학적 요소로 표현

### 인터랙션 철학
- 부드럽고 예측 가능한 움직임 (easing-out)
- Hover 시 미묘한 색상 변화와 스케일 조정
- 스크롤 시 요소들이 부드럽게 페이드인 (dissolve)
- 버튼 클릭 시 물리적 반응성 (scale down 100ms)

### 애니메이션 가이드
- 모든 전환: 180-200ms ease-out
- 카드 호버: 스케일 1.02 + 그림자 강화
- 섹션 진입: 스크롤 시 opacity 0→1, transform translateY(20px)→0
- 배경 요소: 매우 느린 부동(float) 애니메이션 (6-8초)

### 타이포그래피 시스템
- **Display**: Pretendard Bold (900) - 메인 슬로건, 섹션 제목
- **Heading**: Pretendard SemiBold (600) - 카드 제목, 강점 제목
- **Body**: Pretendard Regular (400) - 본문 텍스트
- **Caption**: Pretendard Medium (500) - 라벨, 작은 텍스트
- 라인 높이: 1.6 (본문), 1.3 (제목) - 가독성과 우아함의 균형

## 아이디어 2: "Digital Wellness" - 따뜻한 그라데이션 + 유기적 형태

### 디자인 무브먼트
**Organic Modernism with Warm Gradient Aesthetics**
- 자연스러운 곡선과 따뜻한 색감으로 정신건강의 "인간적" 측면 강조
- 디지털 서비스이지만 따뜻하고 접근 가능한 느낌
- 과학과 감정의 균형

### 핵심 원칙
1. **유기적 형태**: 직선보다 곡선, 경직된 그리드보다 자유로운 배치
2. **따뜻한 팔레트**: 주황, 복숭아, 라벤더 톤으로 감정적 안정감
3. **흐름의 시각화**: "흐름을 기록하고"라는 슬로건을 곡선과 파동으로 표현
4. **접근성 있는 현대성**: 너무 차갑지 않은 SaaS 느낌

### 색상 철학
- **Primary**: 따뜻한 라벤더-퍼플 (oklch(0.65 0.12 290)) - 창의성, 신뢰, 안정성
- **Secondary**: 부드러운 복숭아 (oklch(0.78 0.08 45)) - 따뜻함, 인간성
- **Accent**: 민트-세이지 (oklch(0.72 0.08 160)) - 신선함, 성장
- **Background**: 크림색 그라데이션 (oklch(0.98 0.001 70)) - 따뜻한 배경
- **감정적 의도**: 따뜻하면서도 전문적, 정신건강의 "인간적 회복"

### 레이아웃 패러다임
- 곡선 구분선(SVG divider)으로 섹션 연결
- 카드들이 겹쳐지는 오버래핑 배치
- 비대칭 배치로 자연스러운 흐름 생성
- 배경에 부드러운 그라데이션 + 노이즈

### 시그니처 요소
1. **곡선 구분선**: 각 섹션 사이 부드러운 파도 모양 SVG
2. **블롭 배경**: 추상적인 유기 형태로 배경 채우기
3. **그라데이션 텍스트**: 제목에 미묘한 그라데이션 적용

### 인터랙션 철학
- 부드럽고 유동적인 움직임 (ease-in-out)
- Hover 시 색상 변화와 함께 약간의 회전
- 스크롤 시 요소들이 부드럽게 떠오르는 느낌
- 마우스 움직임에 따라 배경 요소가 미묘하게 반응

### 애니메이션 가이드
- 모든 전환: 200-250ms ease-in-out
- 카드 호버: 스케일 1.03 + 회전 1-2도 + 그림자 강화
- 섹션 진입: 스크롤 시 opacity 0→1, transform rotate(-2deg)→0
- 배경 요소: 느린 부동 애니메이션 (8-10초)

### 타이포그래피 시스템
- **Display**: Noto Serif KR Bold (700) - 메인 슬로건 (세리프로 우아함)
- **Heading**: Pretendard SemiBold (600) - 섹션 제목
- **Body**: Pretendard Regular (400) - 본문
- **Caption**: Pretendard Medium (500) - 라벨
- 라인 높이: 1.7 (본문), 1.4 (제목) - 따뜻하고 여유로운 느낌

## 아이디어 3: "Research-First" - 모노크롬 + 데이터 비주얼라이제이션

### 디자인 무브먼트
**Academic Minimalism with Data-Driven Aesthetics**
- 연구 기반 프로젝트임을 강조하는 과학적 디자인
- 데이터 시각화와 정보 구조를 미학으로 승화
- 대학원 프로젝트의 엄밀함과 혁신성 표현

### 핵심 원칙
1. **정보 우선**: 모든 디자인이 정보 전달을 위해 봉사
2. **모노크롬 우아함**: 검정, 흰색, 회색만 사용하되 깊이감 있게
3. **데이터 비주얼라이제이션**: 차트, 그래프, 다이어그램이 주요 시각 요소
4. **과학적 정밀함**: 픽셀 퍼펙트, 정확한 정렬, 명확한 계층구조

### 색상 철학
- **Primary**: 딥 차콜 (oklch(0.25 0.01 0)) - 신뢰, 권위, 과학성
- **Secondary**: 라이트 그레이 (oklch(0.85 0.01 0)) - 보조 정보
- **Accent**: 차가운 회색-블루 (oklch(0.5 0.02 260)) - 강조 (매우 제한적)
- **Background**: 순백 (oklch(0.99 0 0)) - 명확성
- **감정적 의도**: 신뢰감 있고 과학적, 혁신적이면서도 접근 가능

### 레이아웃 패러다임
- 엄격한 컬럼 기반 그리드 (12-column)
- 데이터 시각화가 중심이 되는 배치
- 텍스트와 비주얼의 명확한 구분
- 여백을 통한 호흡감

### 시그니처 요소
1. **데이터 시각화**: 간단한 라인 차트, 네트워크 다이어그램
2. **구조적 라인**: 얇은 회색 라인으로 정보 구분
3. **숫자 강조**: 통계, 수치를 큰 타이포로 강조

### 인터랙션 철학
- 빠르고 정확한 움직임 (ease-out, 150ms)
- Hover 시 색상 반전 또는 강조
- 클릭 시 즉각적인 피드백
- 데이터 시각화에 마우스 오버 시 상세 정보 표시

### 애니메이션 가이드
- 모든 전환: 150-180ms ease-out
- 카드 호버: 배경 색상 변화 (흰색→라이트 그레이)
- 섹션 진입: 스크롤 시 opacity 0→1
- 데이터 차트: 로드 시 애니메이션 (선 그리기, 막대 상승)

### 타이포그래피 시스템
- **Display**: IBM Plex Mono Bold (700) - 메인 슬로건 (모노스페이스의 과학성)
- **Heading**: Pretendard SemiBold (600) - 섹션 제목
- **Body**: Pretendard Regular (400) - 본문
- **Code/Data**: IBM Plex Mono Regular (400) - 숫자, 통계
- 라인 높이: 1.6 (본문), 1.2 (제목) - 명확하고 정확한 느낌

## 선택된 디자인: "Structured Serenity"

**선택 이유:**
- "흐름을 기록하고, 구조를 만들다"는 슬로건과 가장 잘 맞음
- 정신건강 전문가를 대상으로 신뢰감과 전문성을 동시에 전달
- 미니멀하면서도 구조적 우아함으로 "구조를 만든다"는 의미 시각화
- SaaS 느낌을 유지하면서도 의료적으로 무겁지 않은 톤
- 차분한 연구 기반 서비스 느낌에 가장 부합

**디자인 핵심 가이드:**
- 색상: 딥 네이비(#1a2a4a) + 소프트 민트(#a8d5ba) + 따뜻한 베이지(#e8dcc8)
- 타이포: Pretendard (고딕) - 굵은 가중치로 구조적 느낌
- 레이아웃: 비대칭 모듈식, 충분한 여백
- 애니메이션: 180-200ms ease-out, 부드럽고 예측 가능한 움직임
- 특별 요소: "+"를 기하학적 요소로 표현, 구조적 라인 활용
